## Comment choisir ton cas d'usage ?

### 1. Évaluer le potentiel

| Critère              | Score 1-5 |
| -------------------- | --------- |
| Volume données       |           |
| Répétitivité         |           |
| Impact business      |           |
| Complexité (inversé) |           |

Si score total > 15 : Vaut le coup d'explorer

### 2. Commencer petit

- POC 2-4 semaines
- Cas d'usage simple
- Mesure ROI claire

### 3. Scaler progressivement

- Automatiser 1 processus
- Documenter les résultats
- Étendre aux autres processus
