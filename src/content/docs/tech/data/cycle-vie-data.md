### Sources de Données
- Données internes (CRM, ERP...)
- Données externes (APIs, Web Scraping...)
- Données utilisateurs
- IoT et capteurs

### Méthodes de Collecte
- Automatisée vs. Manuelle
- Temps réel vs. Batch
- Structurée vs. Non structurée
